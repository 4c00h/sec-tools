docker-metasploit
==================

Docker image for Metasploit Framework

Generate image:

		$ git clone git@github.com:infoslack/docker-metasploit.git
		$ cd docker-metasploit
		$ docker build -t docker-metasploit .

Usage with my image:

		$ docker run --rm -it infoslack/docker-metasploit
